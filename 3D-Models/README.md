## 3D-Models
3D-Models is a AI Powered Website made using vite and three.js

## Tech Stack

**Client:** 
React JS

**Server:**
Node JS

  
## Run Locally

Clone this repository

Go to the directory

```bash
  cd 3D-Models
```

Install dependencies

```bash
  cd client/
  npm install
```

Start the Client

```bash
  //open now terminal
  cd client
  npm run dev
```
